# Corresponding Source

Every official binary release of ChanSort Mac must be accompanied, at the same download location and at no additional charge, by a source archive named:

```text
ChanSort-Mac-<version>-Source.zip
```

That archive is the Corresponding Source for the matching application archive and contains the preferred form for modification, including Swift sources, C# sources, the vendored ChanSort loader source, localization, resources, tests, legal notices and build/release scripts.

The required build tools are generally available development tools and are not part of the Corresponding Source: macOS 13 or newer, Xcode/Apple Command Line Tools, Swift 5.9 or newer and .NET SDK 8. Exact package versions are resolved by the project files and documented in `THIRD-PARTY-NOTICES.md`.

Do not publish a binary without its matching source archive. If a mirror separates the files, it must place a clear source link next to the binary and keep the source available for as long as the binary is offered. The publisher is responsible for that availability.
