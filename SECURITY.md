# Security policy

Channel-list files are untrusted input. Please report crashes, path traversal, unsafe archive handling, data loss or arbitrary-code execution privately through the contact options at https://www.pcffm.de before publishing exploit details.

Keep an original device export and verify a backup before writing a modified list to a television. The software is provided without warranty under GPLv3; responsible reports are nevertheless appreciated.

Only the newest published version is supported with security fixes.
